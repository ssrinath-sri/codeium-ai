# JSON Rule Engine TypeScript Project

This is a TypeScript project that demonstrates the use of a JSON rule engine using the `json-rules-engine` library.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Build the project:
   ```bash
   npm run build
   ```

3. Run the example:
   ```bash
   npm start
   ```

4. For development (with ts-node):
   ```bash
   npm run dev
   ```

## Description

The project includes a simple example where a rule checks if the temperature is greater than 100 and triggers an alert event.

You can extend this by adding more rules, conditions, and facts as needed.
